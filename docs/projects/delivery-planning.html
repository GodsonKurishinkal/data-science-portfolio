Packing Strategies

Multiple strategies for different optimization goals

Best Fit

Maximizes space utilization by finding the tightest fit for each box at available extreme points.

First Fit

Fast placement using first valid position found. Good for real-time applications with speed priority.

Sequence Aware

LIFO-optimized loading ensuring last-loaded items are first accessible for delivery order.

Weight Balanced

Distributes weight evenly across the container for vehicle stability and fuel efficiency.

Quick Start

Pack boxes into a delivery truck with sequence awareness

demo.py Python 
from src.packing import BinPacker, Box, BoxType, Container, ContainerType
from src.packing import PackingStrategy, SortingCriterion

# Create boxes with delivery sequence
boxes = [
    Box(id="PKG001", length=40, width=30, height=30, 
        weight=10, sequence=1, box_type=BoxType.STANDARD),
    Box(id="PKG002", length=50, width=40, height=35, 
        weight=15, sequence=2, box_type=BoxType.FRAGILE),
    Box(id="PKG003", length=30, width=25, height=20, 
        weight=5, sequence=3, box_type=BoxType.STANDARD),
]

# Create a delivery truck
truck = Container.from_type(
    ContainerType.LARGE_TRUCK, 
    container_id="TRUCK-001"
)

# Configure packer with sequence-aware strategy
packer = BinPacker(
    strategy=PackingStrategy.SEQUENCE_AWARE,
    sorting=SortingCriterion.SEQUENCE_ASC
)

# Pack and get results
result = packer.pack(boxes, truck)

print(f"Packed: {result.num_packed}/{len(boxes)} boxes")
print(f"Utilization: {result.utilization:.1f}%")

Supported Box Types

Handle different cargo types with appropriate constraints

Standard

Regular packages with no special handling requirements. Can be stacked and rotated freely.

Fragile

Delicate items that must be placed on top. Cannot have heavy items stacked above.

Hazardous

Dangerous goods requiring separation and special placement constraints.

Perishable

Temperature-sensitive items needing priority delivery and specific container zones.

Heavy

Dense items that must be placed at the bottom. Contributes to weight balancing.
