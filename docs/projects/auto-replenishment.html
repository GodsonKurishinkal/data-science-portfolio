ABC-XYZ-FMR Classification

Three-dimensional classification combining value, variability, and frequency for 27 unique segments

ABC (Value)

  • A: 80% of value (top 20% SKUs)
  • B: 15% of value (next 30%)
  • C: 5% of value (bottom 50%)

XYZ (Variability)

  • X: CV < 0.5 (stable demand)
  • Y: 0.5 â‰¤ CV < 1.0 (variable)
  • Z: CV â‰¥ 1.0 (highly erratic)

FMR (Frequency)

  • F: > 70% days with sales
  • M: 30-70% days with sales
  • R: < 30% days with sales

Configuration Example

Define replenishment policies in simple, human-readable YAML files

policies/dc_to_store.yaml YAML 
# DC to Store Replenishment Policy
scenario: dc_to_store
description: "Store replenishment from distribution center"

classification:
  abc:
    A_threshold: 0.80
    B_threshold: 0.95
  xyz:
    X_cv: 0.5
    Y_cv: 1.0

policies:
  AXF:
    method: "continuous_review"
    review_period: 1
    safety_stock_days: 7
    service_level: 0.99
    
  BYM:
    method: "periodic_review"
    review_period: 7
    safety_stock_days: 14
    service_level: 0.95

constraints:
  min_order_qty: 1
  max_shelf_capacity: true
  round_to_case_pack: true
