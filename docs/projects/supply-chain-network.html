Optimization Algorithms

State-of-the-art operations research techniques for logistics network design

Facility Location (CFLP)

Capacitated Facility Location Problem using MILP. Determines optimal DC locations considering fixed costs, capacity, and customer demand.

Vehicle Routing (VRP)

Capacitated VRP with time windows using OR-Tools CP-SAT solver. Optimizes delivery routes for a fleet of vehicles.

Traveling Salesman (TSP)

Classic TSP for single-vehicle route optimization. Foundation for understanding VRP complexity.

Multi-Echelon Inventory

Risk pooling across network tiers. Optimizes inventory positioning at DC vs. store level.

Cost-Service Tradeoff

Pareto frontier analysis balancing logistics cost against delivery speed and service levels.

Geospatial Visualization

Interactive Folium maps showing facility locations, customer zones, and optimized routes.

Quick Start

Solve vehicle routing problems with Google OR-Tools in Python

vrp_solver.py Python 
from ortools.constraint_solver import routing_enums_pb2
from ortools.constraint_solver import pywrapcp

# Create routing model
manager = pywrapcp.RoutingIndexManager(
    len(distance_matrix),
    num_vehicles,
    depot_index
)
routing = pywrapcp.RoutingModel(manager)

# Distance callback
def distance_callback(from_idx, to_idx):
    from_node = manager.IndexToNode(from_idx)
    to_node = manager.IndexToNode(to_idx)
    return distance_matrix[from_node][to_node]

transit_callback_index = routing.RegisterTransitCallback(distance_callback)
routing.SetArcCostEvaluatorOfAllVehicles(transit_callback_index)

# Add capacity constraint
routing.AddDimensionWithVehicleCapacity(
    demand_callback_index,
    0,  # null capacity slack
    vehicle_capacities,
    True,
    'Capacity'
)

# Solve
search_params = pywrapcp.DefaultRoutingSearchParameters()
search_params.first_solution_strategy = (
    routing_enums_pb2.FirstSolutionStrategy.PATH_CHEAPEST_ARC
)
solution = routing.SolveWithParameters(search_params)

Key Insights

Practical learnings from optimizing distribution networks across scenarios

Consolidation Benefits

Fewer, larger DCs reduce fixed costs but increase transport. Optimal is typically 3-5 regional hubs for national coverage.

Time Windows

Adding delivery windows increases route distance by 10-15% but enables better customer service and dock scheduling.

Fleet Sizing

Optimal fleet balances vehicle costs against utilization. 80-85% utilization is typically ideal.

Geographic Clustering

Pre-clustering customers by zone before VRP improves solution quality and computation time by 30-40%.
