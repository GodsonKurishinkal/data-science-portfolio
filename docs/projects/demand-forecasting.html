Business Problem

S&OP processes fail when forecasts are inaccurate or generated too late for planning cycles. This system addresses the core challenge: providing demand forecasts at the right granularity (product-store-day) with enough lead time to inform procurement decisions.

The ensemble approach achieves 12.3% MAPE‚Äîa critical metric for balancing inventory investment against service level targets. Processing 30,490 time series across 10 stores, 3 states, and 3,049 products.

Quick Start

Get up and running in under 3 minutes with this simple pipeline

demo.py Python 
# Run the complete ML pipeline in under 3 minutes
from src.data.preprocessing import preprocess_m5_data
from src.features.build_features import build_m5_features
from src.models.train import train_m5_model

# Load and preprocess data
df = preprocess_m5_data(data_path='data/raw')

# Engineer 50+ features
df_features = build_m5_features(df, target_col='sales')

# Train LightGBM model
model, metrics, importance = train_m5_model(
    df_features,
    model_type='lightgbm',
    test_size=0.2
)

# Display results
print(f"R¬≤ Score: {metrics['r2']:.4f}")
print(f"MAPE: {metrics['mape']:.2%}")

Key Insights

Discoveries from analyzing 58M+ data points across 30,490 time series

Temporal Patterns

Sales lag features (7 and 28 days) are the strongest predictors, indicating clear weekly and monthly seasonality.

Price Sensitivity

Price changes significantly impact sales, with elasticity varying by category. FOODS show less sensitivity than HOBBIES.

Event Impact

Special events drive 20-40% sales spikes, with cultural events showing regional variation.

Cost Savings

Potential 10-15% reduction in inventory costs through optimized stock levels.
