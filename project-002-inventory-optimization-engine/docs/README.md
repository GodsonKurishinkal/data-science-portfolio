# Documentation

## Overview

This directory contains documentation for the Inventory Optimization Engine.

## Contents

- **[MODEL_CARD.md](MODEL_CARD.md)** - Detailed model documentation
- **[images/](images/)** - Visualization outputs and diagrams

## Visualizations

Key visualizations generated by the system:

1. **ABC-XYZ Matrix** - Heatmap showing item distribution
2. **Cost Breakdown** - Pie/bar charts of inventory costs
3. **Service Level Tradeoff** - Cost vs. service level curves
4. **Inventory Turnover** - Turnover analysis by class
5. **EOQ Analysis** - Order quantity optimization curves
6. **Safety Stock Distribution** - Histogram of safety stock levels

## Generating Documentation

To regenerate documentation:

```bash
# Generate visualizations
python scripts/generate_visualizations.py

# Build Sphinx docs (if configured)
cd docs/
make html
```
